Step 1: Find the discriminant, D = b² − 4ac.

For 2x² + 5x − 3 = 0 we have a = 2, b = 5, c = −3, so D = 25 − 4(2)(−3) = 25 + 24 = 49.

Step 2: Since D > 0, there are two real, distinct roots.

Step 3: Apply the quadratic formula. Taking the square root, √49 = 7, so x = (−5 ± 7) / 4, giving x = ½ and x = −3.

We can verify: 2(½)² + 5(½) − 3 = ½ + 2½ − 3 = 0 ✓ and 2(−3)² + 5(−3) − 3 = 18 − 15 − 3 = 0 ✓…

Quadratic Equations — Roots and the Discriminant
A quadratic equation has the general form ax² + bx + c = 0, where a ≠ 0. Its two roots are given by the quadratic formula:
x=−b ± √b² − 4ac2a
The quantity D = b² − 4ac is called the discriminant. Its sign decides the nature of the roots:
Discriminant Nature of roots Example
D > 0Two real, distinct rootsx² − 5x + 6 = 0
D = 0One real, repeated rootx² − 4x + 4 = 0
D < 0No real rootsx² + x + 1 = 0
In code, the same check comes first, so we never take the square root of a negative number:
import math def roots(a, b, c): d = b*b - 4*a*c # discriminant if d < 0: return None # no real roots r = math.sqrt(d) return ((-b + r) / (2*a), (-b - r) / (2*a))
